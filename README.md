# Roblox AI Anti-Cheat

A behavior-based anti-cheat system for Roblox using machine learning (LSTM & GRU Autoencoders)

## 🚀 Features
- Player movement data collection (Roblox)
- CSV export for analysis
- Modular script structure (server, client, shared)
- Planned AI-based anomaly detection

## 🛠️ Tech Stack
- Roblox (Luau)
- Rojo (VS Code sync)
- Python (for ML, upcoming)

## ⚠️ Status
This project is currently in development
Machine learning pipeline and detection system are in progress
